📝 Description

Ce projet consiste en la création d'un tableau de bord interactif d'analyse des ventes. L'objectif était de centraliser les indicateurs clés de performance (KPIs) pour permettre une prise de décision rapide sur la performance des produits, des régions et de la force de vente.

Le dashboard permet de filtrer les données par mois et offre une vue granulaire sur la rentabilité et les tendances de ventes sur une période de deux ans (2023-2025).

🛠️ Compétences & Outils Utilisés

Analyse de Données : Nettoyage des données, calcul des marges de profit et agrégation des ventes.

Visualisation : Création de graphiques complexes (Heatmap géographique, diagrammes en barres, secteurs et séries temporelles).

KPIs Clés : Suivi du Chiffre d'Affaires total, identification du produit phare (Chair) et calcul du profit total.

Outils : Excel / Google Sheets (Tableaux croisés dynamiques, fonctions avancées, mise en forme conditionnelle).

📈 Fonctionnalités du Dashboard

Analyse Temporelle : Visualisation de l'évolution des ventes sur plusieurs années pour identifier les saisonnalités.

Performance Géographique : Carte interactive des États-Unis montrant la répartition des ventes par État.

Top Performance : Identification automatique du meilleur vendeur et du produit ayant généré le plus de revenus.

Détail des Transactions : Table de données structurée permettant de vérifier chaque ligne de vente avec son mode de paiement associé.

💡 Insights Clés (Exemples)

Le produit "Chair" a été identifié comme le moteur principal des ventes pour le mois sélectionné.
